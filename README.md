# Personal Site Starter

A no-build personal site that runs on GitHub Pages. Uses Tailwind via the Play CDN so you can push and go.

## Quick start

1. Create a repository named `<your-username>.github.io` on GitHub.
2. Download `personal-site-starter.zip` from our chat and unzip it.
3. Drag all files into the new repo and commit.
4. Visit `https://<your-username>.github.io` after a minute.

## Local preview

Open `index.html` in your browser. No tooling required.

## Customize

- Edit text in `index.html`.
- Replace links with your profiles.
- To add posts, link to Markdown files or external URLs from the Writing section.
- Add a `CNAME` file if you use a custom domain.
